# webpack5-react-typescript-template
REACT template for frontend project

## Getting started
Follow the instructions to set-up and run this app.

### Prerequisities
1. Node(>=v13.3.0)
2. yarn(>=1.15.2)

### Steps to run app on local
```
yarn install
yarn compile
yarn start
yarn dev:build
yarn dev
yarn prod:build
yarn prod
```
